# 疫情播报小助手

1月末，疫情初期在家的练手项目。当时的目的是提醒自己关注疫情状况，数据源来自网易，当然现在各种各样的疫情播报，再加上疫情国内已经到达末期，项目不在维护。

#### 功能：

- 通过邮箱定时发送相关疫情信息
- 支持邮箱回复关键字，查询地区的具体状况，例如：回复山东，查询山东各省市状况
- html邮箱定制信息
